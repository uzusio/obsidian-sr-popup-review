# Popup Review for Spaced Repetition

[English README](README.md)

Obsidianを開かなくても [Spaced Repetition](https://github.com/st3v3nmw/obsidian-spaced-repetition) の暗記カードを復習できるプラグインです。一定間隔でデスクトップ右下に小さな最前面ポップアップが現れ、その場で「問題を読む → 答えを見る → Again / Hard / Good / Easy で評価」まで完結します。

評価は Spaced Repetition プラグイン**自身のレビュー処理**（レビューモーダルのボタンと同じコードパス）を通して書き込まれるため、スケジュールデータの整合性は本家と完全に同一です。SM-2 / FSRS の再実装はしていません。

Spaced Repetition **v1.15.4** で動作確認しています。

## 動作の流れ

1. 設定した間隔（デフォルト2時間）ごとに、期限が来ているカードがあるかチェックします。
2. あれば、プライマリディスプレイ右下にポップアップが出ます。**フォーカスは奪いません**。Obsidianは最小化したままでOKです。
3. 問題を読み、**答えを見る**（または `Space`）で答えを表示し、4つのボタン（または `1`〜`4` キー）で評価します。
4. 評価は Spaced Repetition 経由で保存され、ポップアップは閉じます。評価せずに閉じた場合は何も書き込まれず、カードは期限のまま残ります。

## 必要環境

- Obsidian **デスクトップ版**（Windows / macOS / Linux。ポップアップにElectronウィンドウを使うためモバイル非対応）
- [Spaced Repetition](https://github.com/st3v3nmw/obsidian-spaced-repetition) プラグイン（インストール済み・有効）

## インストール

### コミュニティプラグインブラウザから（推奨）

1. 設定 → コミュニティプラグイン → 閲覧 を開く
2. 「**Popup Review**」で検索して **Popup Review for Spaced Repetition** をインストール
3. 設定 → コミュニティプラグイン で有効化

### 手動

1. [最新リリース](https://github.com/uzusio/obsidian-sr-popup-review/releases) から `main.js` / `manifest.json` / `styles.css` をダウンロード
2. `<あなたのvault>/.obsidian/plugins/sr-popup-review/` に配置
3. Obsidianをリロードして **Popup Review for Spaced Repetition** を有効化

## ポップアップの使い方

| 操作 | 効果 |
| --- | --- |
| **答えを見る** をクリック / `Space` か `Enter` | その場で答えを表示 |
| **Again / Hard / Good / Easy** をクリック / `1`〜`4` キー | 評価を保存して閉じる |
| **✕** をクリック / `Esc` | 何も保存せずに閉じる |
| ヘッダをドラッグ | ポップアップを移動 |

評価ボタンのラベルは Spaced Repetition 側で設定したものがそのまま使われます。カード本文はマークダウンとして描画され、穴埋め（cloze）のマスクも通常のレビューモーダルと同じです。

コマンドパレットの **「今すぐレビューポップアップを表示」** で、任意のタイミングでポップアップを出すこともできます。

自動ポップアップを一時停止したいときは、**右下ステータスバーのベルアイコン**、コマンド **「ポップアップの一時停止を切り替え」**、設定のトグルのどれでも切り替えられます（3つとも同じスイッチです）。停止中でも手動の「今すぐ表示」コマンドは動きます。

## 設定

| 設定 | デフォルト | 説明 |
| --- | --- | --- |
| 言語 | Obsidianの設定に従う | このプラグインの表示言語（English / 日本語） |
| ポップアップを一時停止 | オフ | 自動ポップアップを一時停止（右下ステータスバーのベルアイコン・コマンドでも切替可） |
| ポップアップ間隔（分） | 120 | ポップアップを出す間隔（最小5分） |
| ポップアップ停止時間帯 | オン・01:00〜09:00 | トグル＋時間範囲の1行UI。この時間帯はポップアップを出さない。日付をまたぐ範囲も可 |
| 自動クローズ（秒） | 90 | 無操作のポップアップを閉じる（何も書き込まない）。0で無効 |
| 期限カードのみ | オン | 期限が来ているカードだけを出す。オフにすると新規カードも出る |
| デッキ順を無視してランダムに出題 | オン | 期限枚数で重み付けしたランダムなデッキから出題（全期限カードがほぼ等確率）。オフでSpaced Repetition本来のデッキ順 |
| デッキフィルタ | 全デッキ | 全デッキ / リストのデッキのみ |
| デッキリスト | — | デュアルリスト形式：「既存のデッキ」⇄「対象のデッキ」を追加/除外ボタン（ダブルクリックも可）で移動。対象デッキはサブデッキにも適用。対象が空の間は全デッキ |
| デッキ名を表示 | オン | ヘッダにデッキパスと期限枚数を表示 |
| 起動直後にチェック | オフ | Obsidian起動の約15秒後に1回チェック |

設定画面にはSpaced Repetition連携の状態（動かない場合はその理由）も表示されます。

## データの安全性

- 評価は Spaced Repetition 自身のレビューシーケンサーを通ります。モーダルのボタンを押した場合と**同一の書き込み**です（スケジュールコメント、兄弟カードのbury、ロードバランス、FSRS/SM-2すべて含む）。
- このプラグインは Spaced Repetition の内部実装に依存するため、**使用する内部要素を毎回すべて検査**します。将来のバージョンで内部が変わった場合は通知を出して機能を停止し、**不明なコードパスで書き込むことは決してありません**。
- ポップアップを閉じた・自動クローズされた場合は何も書き込まれません。

## 既知の制限

- ポップアップが出るのはObsidianの起動中のみ（最小化はOK）
- ポップアップは自前の最前面ウィンドウであり、OSの通知ではありません。通知センターには残らず、集中モード（応答不可）も無視して表示されます
- 表示位置は現状プライマリディスプレイの右下固定です
